# Zepto-data-cleaning-project-
1. This project analyzes an e-commerce dataset containing order details such as product category, city, price, quantity, payment method, and order status.

2. The dataset is cleaned and organized to remove errors and ensure accurate data analysis.

3. Various attributes are examined to understand customer purchasing patterns and delivery performance.

4. The analysis helps in gaining insights that can support better business decisions and improve customer service.
